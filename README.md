### Data
This project uses the **GSE65904 melanoma dataset** from [NCBI GEO](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE65904).  
Annotation file was obtained from the GPL10558 platform on NCBI GEO.
Please download the dataset manually and place the files in the `data/` folder before running the analysis.
